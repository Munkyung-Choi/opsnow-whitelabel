### 📝 Core Logic

`domain_requests` 테이블이 클라우드에만 존재하는 Shadow DB 상태를 역공학 마이그레이션으로 해소.
ENUM + 테이블(11컬럼) + RLS 4개 + SECURITY DEFINER 트리거 함수를 멱등성 보장 SQL로 재현.
`supabase db push` 로 클라우드 `supabase_migrations` 등록 완료, TypeScript 타입 재생성.
