## 📝 Core Logic

1. **Discriminated Union 타입 시스템**: `CurrentUser = MasterAdmin | PartnerAdmin` 로 `partner_id` 타입을 역할별 런타임 가드 대신 컴파일 타임에 강제.
2. **역할별 invariant 화이트리스트**: `Record<UserRole, Validator>` 로 exhaustive 체크 + Silent Fail with Auditing (redirect + 감사 로그, throw 금지).
3. **Open Redirect 3단계 방어**: `validate-next-url.ts` 독립 모듈 — 재귀 디코딩 + NFKC 정규화 + URL 파싱 + 화이트리스트/데니리스트 이중 게이트.
