## 📝 Core Logic

1. **세션 사이드카 패턴**: Supabase auth 세션 불변 유지 + `opsnow_impersonate` 서명 쿠키로 대리 접속 컨텍스트 병기. `system_logs.actor_id`는 항상 master의 `auth.uid()`로 기록되어 감사 무결성 보장.
2. **HMAC-SHA256 서명 쿠키**: `<partner_id>.<issued_at>.<issued_for_user_id>.<kid>.<sig>` 5-segment payload. `timingSafeEqual` 상수 시간 비교, ±5초 clock skew leeway, 1시간 만료.
3. **Race-safe Route Handler**: system_logs INSERT 성공 후에만 쿠키 세팅 + 최근 5초 동일 actor dupe 로그 검사로 동시 POST 방어.

### AP 아키텍트 추가 보강

- ✅ **`timingSafeEqual`** Node crypto Buffer 비교로 사용 (스트링 `===` 사용 금지 원칙 준수)
- ✅ **Clock skew ±5초** `CLOCK_SKEW_MS` 상수로 leeway 허용
- ✅ **맥락 오버레이 UX** 배너에 파트너명 병기 (`현재 {business_name} 계정으로 대리 접속 중입니다`)
- ✅ **IMP-07 초정밀 검증** `action` / `actor_id` / `on_behalf_of` / `target_id` / `target_table` / IP 모두 시스템 로그 E2E에서 직접 조회·검증
