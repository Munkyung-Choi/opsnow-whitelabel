# Decision Journal: 2026-04-21

> git log 기반 소급 작성 — 세션 종료 시 /dev-end 미실행. 2026-04-22 dev-start에서 복원.

## Decisions

- **WL-141 MASTER_UID 조회 — auth.users(listUsers) → profiles 테이블로 교체**: listUsers는
  GoTrue 페이지네이션 이슈와 권한 제약이 겹쳐 E2E에서 MASTER_UID를 안정적으로 찾지 못했다.
  profiles 테이블 직접 조회(email 기반)로 교체하여 불확실성 원천 제거. WL-141이 이 커밋으로
  완결됨.

- **환경변수 자동검증 시스템 도입 — @required 어노테이션 + pre-push gate**: CI에서
  IMPERSONATION_SIGNING_SECRET 누락으로 impersonate API 500 에러가 발생한 사건을 계기로
  선언적 검증 체계를 도입. 코드에 @required로 선언하면 서버 시작 시 env 누락이 즉시 발각되고,
  pre-push vitest gate가 누락된 상태로 push 자체를 막는다. .env.example도 git 추적으로 전환해
  신규 env 추가 시 문서화가 강제됨.

## Rejected Paths

- **listUsers pagination 보완 접근**: MASTER_UID 문제를 listUsers 전체 순회 개선으로 해결하려
  했으나, 근본은 "GoTrue API를 통한 간접 조회"의 구조적 불안정성이었다. profiles 직접 조회로
  레이어를 낮추는 것이 더 단순하고 안전한 방향.

## Debts & Risks

- **IMPERSONATION_SIGNING_SECRET CI 누락 사건 교훈**: env 추가 시 CI secrets 등록을 빠뜨리는
  패턴이 반복되고 있음. @required 체계로 로컬 누락은 잡히지만, CI secrets 누락은 여전히
  사람이 직접 등록해야 하는 한계 존재.
- **레거시 seed 파일 제거** — superseded 파일들 정리 완료. DEBT-005의 부분 상환.
